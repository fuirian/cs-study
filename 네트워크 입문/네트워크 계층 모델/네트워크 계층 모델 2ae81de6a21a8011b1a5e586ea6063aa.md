# 네트워크 계층 모델

### OSI 7계층

- 물리 계층→데이터 링크 계층→네트워크 계층→전송 계층→세션 계층→표현계층→응용계층
- 출발지 컴퓨터의 응용 계층→물리계층→네트워크 통신망→목적지 컴퓨터의 물리계층→응용계층

### TCP/IP 모델

- OSI 7계층 모델과 비슷하지만 4계층으로 이루어짐
- TCP/IP 모델의 응용 계층
    - OSI 7계층의 세션 계층, 표현 계층, 응용계층
    - 전송할 데이터를 준비한 후 응용 계층에서 처리할 각종 부가 정보를 붙여 전송 계층으로 내려 보냄
- TCP/IP 모델의 전송 계층
    - OSI 7계층의 전송 계층
    - 응용계층의 데이터를 받아 전송 계층에서 처리할 부가 정보를 붙여 인터넷 계층으로 내려보냄
- TCP/IP 모델의 인터넷 계층
    - OSI 7계층의 네트워크 계층
    - 전송 계층의 데이터를 받아 인터넷 계층에서 처리할 각종 부가 정보를 붙여 네트워크 인터페이스 계층으로 내려보냄
- TCP/IP 모델의 네트워크 인터페이스
    - OSI 7계층의 물리 계층, 데이터링크 계층
    - 인터넷 계층의 데이터를 받아 네트워크 인터페이스에서 처리할 부가정보를 데이터 앞뒤로 붙여 네트워크 통신망으로 전송
- 각 계층이 만든 데이터 단위
    - 응용 계층 → 데이터
    - 전송 계층 → 세그먼트
    - 인터넷 계층 → 패킷
    - 네트워크 인터페이스 계층 → 프레임

### OSI 7계층 모델과 TCP/IP 모델 비교

- OSI 7계층
    - 실제 네트워크 시스템 구현에 한계
- TCP/IP
    - 단순하고 실용적이라 실제 인터넷에 사용

| **계층** | **OSI 계층 모델** | **TCP/IP 모델** | **설명** |
| --- | --- | --- | --- |
| 7 | 응용 계층 | 응용 계층 | 사용자가 직접 사용하는 프로그램에서 데이터 생성 |
| 6 | 표현 계층 | 응용 계층 | 응용 계층에서 받은 데이터를 적절한 형식으로 변환 |
| 5 | 세션 계층 | 응용 계층 | 두 장치 간 통신 연결 설정 및 관리 |
| 4 | 전송 계층 | 전송 계층 | 데이터를 세그먼트로 분할, 세그먼트가 보낸 순서대로 도착하도록 전송의 신뢰성 보장 |
| 3 | 네트워크 계층 | 인터넷 계층 | 세그먼트를 패킷으로 변환, IP 주소를 이용해 패킷을 최종 목적지까지 전송 |
| 2 | 데이터 링크 계층 | 네트워크 인터페이스 계층 | 패킷을 프레임으로 변환, MAC 주소를 이용해 네트워크 내 장치로 프레임 전송 |
| 1 | 물리 계층 | 네트워크 인터페이스 계층 | 전기신호, 케이블 등을 통해 프레임을 전기 신호나 광신호로 변환해 전송 |
