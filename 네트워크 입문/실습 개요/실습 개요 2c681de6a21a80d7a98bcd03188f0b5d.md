# 실습 개요

### 클라우드 컴퓨팅과 GCP

- **온프레미스**
    - 기업이나 개인이 자체적으로 IT 인프라를 소유하고 운영하는 방식
    - 초기 서버 구축 비용이 많이 들고, 유지보수에 많은 시간과 노력 필요
    - 트래픽이 갑자기 증하가면 유연하게 대처 불가
- **클라우드 컴퓨팅**
    - 온프레미스의 단점을 해결하기 위해 등장
    - 인터넷을 통해 자원을 필요한 만큼 빌려 쓰고 사용한 만큼만 비용 지불
    - **비용 효율성**
    - **운영 유연성**
    - **구축 신속성**
    - **관리 용이성**
    - **글로벌 서비스**
- GCP
    - 구글 계정과 손쉬운 연동
    - $300 무료 크레딧 제공

### 실습 과정

- VM 인스턴스 생성
    - 구글로부터 가상으로 CPU, 메모리, 스토리지 등의 자원을 할당받아 사용하는 방식
    - 실제 존재하는 서버처럼 운영체제를 설치하고 프로그램을 실행해 외부 네트워크와 통신 가능
- SSH 프로토콜을 이용해 원격 접속
    - 안전하게 내 컴퓨터에서 서버에 접속해 시스템 제어 가능
- SFTP 프로토콜로 내 컴퓨터에 있는 파일 전송
    - 웹 페이지를 구성하는 HTML, CSS, JS 파일
- HTTP 프로토콜로 웹 페이지를 인터넷을 통해 다른 사용자가 볼 수 있도록 설정
